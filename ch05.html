Intro to Weights and Setting Up

An Apophysis 2.05 beta tutorial

This tutorial discusses two things. First, it discusses Apophysis Weights, and why you should care about them. Second, it discusses why there is sometimes so much noise in your beautiful Apophysis fractal. By noise, I mean unsightly, stray pixels that get in the way of making a beautiful fractal.

So, what are weights?

Weights tell how visible certain parts of the fractal will be. Let's use Sierpinski's quilt from the last tutorial as an example. Set up Sierpinski's quilt. See The previous chapter if you need help setting it up. If you've already set it up, skip this part.
