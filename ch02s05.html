Set Up Color

Because we reset our colors with the new blank flame button, we need to set them up again. The first thing to do is to select a gradient that we like to put on the fractal.

Go to the main window, and push the gradient button . This will bring up the gradient dialog, as shown below. When you are in the gradient dialog, go to the Preset menu. Scroll down until you find a gradient you like. Julias usually do well with very busy, multicolored gradients. For the fractal in the intro, I used gradient 259-Apophysis040427KaosFish, but feel free to use whichever gradient you want.

Go back to the Transform Editor and select transform 1 . Go to the Colors tab . Make sure the transform color field is set to 0.000 . This will put the color slider is all the way to the left . Select transform 2 . In the colors tab, type in .333 . This will put the color slider a third of the way to the right . Select transform 3 . Type in .666 in the color tab . This will put the color slider two-thirds of the way to the right . Select transform 4 . Type 1.000 in the color tab . This will move the color slider all the way to the right . This will also help make sure that fractal has brilliant colors when the detail of the fractal starts showing up.
