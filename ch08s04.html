What did you just do? Well, that set of weird code and numbers that you copied is called a parameter set. This is how Apophysis stores your fractals. I shall explain them in greater detail in another tutorial.

So, anyway, you know have a fractal that looks like a box. It was created using the sinusoidal variation and post transforms. In order to change the shape of the rectangles, make sure you have the PX button

selected. If it's not selected, you won't be able to edit the post transform.

All right. Select a transform, and make sure that the post transform button is pressed. Now, right-click on the transform. A menu will show up that looks like the menu below. Select duplicate. This will make a duplicate of the transform right on top of the selected transform. Select this duplicate and drag it around. Hey! You got a new sinusoidal!

You can also copy and paste transforms by hitting the duplicate button

at the top, or by hitting control-c and control-v.

So, by copying and pasting these transforms, you can create things like this impossible triangle.

Now, add a linear transform by going up and pressing the add transform button.

It will create a new transform over the reference triangle with a linear variation of 1. Change this triangle's weight to 20. This will make the copies of the box more visible. Now, take this triangle and move it around.

You can make spirals out of the boxes, you can make a line of boxes, you can do a whole bunch of fun things with the boxes! If you add another linear transformation, you can tile the boxes, too.

Post transforms are fairly easy to understand. Here's a few more miscellaneous tips and tricks. First, you can't add variations to a post transform. (At least, you can't yet. Further versions of Apophysis might change this). If you change the variation when working with a post transform, it will change the variation on the original transform. Also, using post transforms on blur and bubble transforms leads to some more interesting fractals.

Finally, you can create some really neat fractals if you have a basic knowledge of geometry and trigonometry. The most difficult part of the whole process is getting the angles of the parallelograms perfect. I used trigonometry to calculate the angles for the box fractal, and posted the parameters here. However, it is an interesting challenge to do it on your own. I may write an introduction to trigonometry tutorial, but if and when I do, it won't be in the basic tutorials section.

So that's it for post transforms. Here's a challenge for you: write your name in boxes. And, until next time, have fun!
