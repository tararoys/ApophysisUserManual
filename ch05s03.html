Unbalanced Weights (And Why They Cause Noise)

So, lets take a look at a fractal where the weighs vary wildly. Set the red transform's weight to 1. Set the yellow transform's weight to .5. Set the green transform's weight to .2, and set the blue transform's weight to .1

Also, note that the less visible parts of the fractal are very 'noisy.' This means that there are a lot of stray pixels, making the picture look dotty and corrupted. It doesn't look good. If you have a lot of stray pixels, you might want to try playing with the weights. Remember, weights who's values are close to each other create smooth fractals, while wildly varying weights create noise.
