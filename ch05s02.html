Weights: Making Parts of the Fractal More Visible

Click on the red transform. Under the transforms tab, set its weight to 1. Click on all of the other transforms, and set their weights to .2. Look at the fractal.

Notice how the part of the fractal the red triangle is responsible for suddenly became much more visible.

Go back to the red transform and set its weight to .2. Now look at the fractal. All parts of it are equally visible, right? Right.

Now set the red transform's weight to .05. Notice how the part of the fractal it is responsible for suddenly became less visible. Interesting, huh?

Finally, set the red transform's weight equal to zero. Wow! It's no longer a Sierpinski's quilt! Instead, it's a Sierpinski's triangle! Also note that the weight is no longer set to zero- it's set to a very very small numbers. Apophysis does not like weights set to zero, so it doesn't allow them.

So what does this all mean? It means that the value of the weight (and by value I mean the number the weight is set to) isn't important. What's important how high or low the value of the weight is compared to the weights of the other transforms. For example, when we set the red transform's weight high in comparison to the rest of the transform, that part of the fractal became much more visible. When we set the red transform's weight low in comparison to the weight of the other transforms, it became much less visible. And when we set that weight to zero, disappeared entirely. So, repeat after me: the value of the weight is not important. The important thing is the value of the transform compared to the value of the weights of the other transforms.
