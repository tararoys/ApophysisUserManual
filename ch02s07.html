Moving Transforms 2: Making a Diamond Shape

Arrange the transforms in a diamond as shown above. You don't have to arrange the transforms so that the colors match up. The algorithm that makes fractals work does not care what color a transform is, or what number it is. It only cares about the position the transform is in and the variations added to that transform. However, but it does make this tutorial a lot easier if you use the same transforms I did. I will be telling you to do things to a transform of a specific color, because it is easier to say "put a linear variation of 1 on the red triangle" instead of "put a linear variation of 1 on the triangle who's O coordinate is (0, 0), whose X coordinate is (0, 1) and whose Y coordinate is (1,0)." This tutorial will be assuming that you matched the colors.

The exact coordinates of the diamond are shown above, if you want to type them in by hand. I recommend this, because the more precise you are, the clearer your Julia will be at the end.

If you wish to cheat, simply copy the parameter set below, and when you are in the Apophysis Main Window, go to the Edit menu and select Paste. This will paste the fractal below into Apophysis.
