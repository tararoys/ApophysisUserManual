Using Apomap

ApoMap is a free program that can be downloaded from the Apohysis website under the ApoMap Palette Editor link. It allows you to quickly and easily make your own Apophysis gradients.

First of all, open Apophysis. For an introduction to gradients and gradient editing, check out basic tutorial 3.

Copying A Gradient

Anyway, open the gradient tab. It looks like the picture below.

There is a button called the copy gradient button. Choose a gradient that you like, then hit the copy gradient button.

This will copy a gradient onto the clipboard so that you can paste it into a program like Apomap.

Create a New Gradient

Next, open Apomap. You will see a screen that looks like the screen below.

Go to file and select new. This will create a new gradient.

Now press the paste gradient button.

This will paste the gradient that you copied from from Apophysis into Apomap. Do not confuse the paste gradiet button

with the paste button.

They look similar, but they do two different things. You should now have a window that looks like the window below.

Process Menu Options

All right. Now we can start seriously playing with colors. In the gradient window you will see a blue bar running across the top of your gradient.

Now, click and drag your mouse across part of the gradient. Note how the blue bar corresponds to the area that you clicked and dragged on. This means that that part of the gradient is selected. When we begin to use buttons to manipulate the gradient, only this part of the gradient will be affected.

All right. Now we'll go over what various options do. The most important menu in ApoMap is the process menu. This is what will let you manipulate the gradient. So, open up the process menu, and we'll go through what the various options do.

Randomize

Randomize: This affects the whole gradient, no matter what part you have chosen. Essentially, it makes a gradient out of random colors that generally look good together.

Smoothize

Smoothize: This also affects the whole gradient, and I'm not exactly sure what it does. But it takes colors from the whole gradient and make smooth bands of separate colors.

Grayscale

Grayscale: This affects only the selected part of the gradient. It makes the select part of the gradient gray.

Invert

Invert: Invert switches all of the colors in the gradient to their complements. For example, red colors switch to greens, yellow colors switch to purple, and blue colors switch to orange.

Reverse

Reverse: Reverse switches the order of the colors in the selected area. In essence, it flips that section of the gradient horizontally.

Blur

Blur: Blur makes the bands of color in the gradient less distinct. It only affects the selected area.

RGB

Blend RGB: This will only affect the selected area. This is a rather odd option. It takes the color at the beginning of the selection and the color from the end of the selection and makes a gradient blending the two together through the selected area. See picture below for a better idea of what I'm talking about.

Blend HSV

Blend HSV: I'm not sure what this function does. It only works on the selected area, and it is simply another way of blending the gradient.

Random Blend

Random Blend: This picks two random colors and blends them together.

ROL and ROR

ROL and ROR: These to strange things to the selected portion of the gradient. I'll let you play with them to figure out exactly what. I will say that these are generally not very useful options.

Adjust Editor

Adjust: Now we come to one of the most useful tools in Apomap: the adjust editor You can access it from the process menu or by clicking the adjust editor button.

When you click on this option, a dialog box comes up, with several sliders, like the picture below.

So what do all these options do? Let's take them individually. These options only work on the selected area of the gradient. I'll show what each of these options do to the sample gradient, shown below.

Hue

Hue: This changes the colors of the selected area.

Saturation

Saturation: This changes how vibrant or dull the colors are. Moving the slider to the right makes the colors more vibrant. Moving the colors to the left makes the colors duller.

Brightness

Brightness: This changes how bright or dark the gradient is.

Contrast

Contrast: This changes how bright or dark the colors appear in relation to each other.

Red, Green, and Blue

Red: This changes how much red is in the gradient. When the slider is moved right, more red is added. When the slider is moved left, red is taken away.

Green: This changes how much green is in the gradient. When the slider is moved right, more green is added. When the slider is moved left, green is taken away.

Blue: This changes how much blue is in the gradient. When the slider is moved right, more blue is added. When the slider is moved left, blue is taken away.

Blur

Blur: This changes how clearly the seperate lines show up in the gradient.

Frequency

Frequency: This changes how often the pattern of the gradient repeats itself.

Rotation

Rotation: This changes where the colors are positioned in the gradient.

Play with all of these functions to get a feel for how they work. They are, far and away, the most useful functions in Apomap.

The Graph Editor

Next is the graph function editor. You can access it from the process menu or by pressing the graph button.

It's not a very useful editor, but it is fun to play with and is also an interesting representation of the color data. So, open the graph editor. It will, in general, look like the screen below.

Look at the gradient you are working on. In this tutorial, I am working on this gradient.

Note that the gradient consists of a bunch of stripes of different colors. Each of those stripes is one pixel wide. There are 256 of these stripes, which means that when you put this gradient in Apophysis, you can color your fractal with 256 different colors. Now, look at the graph. You will see a red, green and a blue line. The position of these lines corresponds to how much red, green and blue there is in any given stripe. Combining the amounts of red, green, and blue produces the color of that stripe. It's rather like how a painter mixes colors in order to get new ones.

Allow me to give a small example. Look at the graph below. At the top of the graph, you will see a pair of numbers. Run your mouse over the graph. The numbers change. What do these numbers stand for? The first number is the number of the one-pixel-wide stripe of color on the gradient. The second number is the amount of color that that stripe would have if you were to click on that location. So, look at the picture below. My mouse is hovering over the green line at stripe 128. Look at the second number. This means that the amount of green in that stripe is set to level 178, which is a little over 2/3 of the amount of total amount of green that can be added in this program. In other words, it's a lot of green.

You might ask why you can only set a color to a maximum of 255. This is a good question, and the answer involves explaining hexedecimal code and how colors are represented by a computer. If you're really interested in the topic, all is explained in part 2 of this tutorial.

Anyway, that's what the numbers stand for. So how do we actually use this graph? It's fairly simple. Select one of these three color buttons,

depending on what color you want to change. You can now edit that color's line. Say, for example that you chose blue. Now, click on the graph. A line will appear from your mouse to the point you clicked on.

Now, click again. The graph changes so that it followed your line. Your gradient also changes. In this case, since we upped the red in a certain section, that section turned orange.

Now you know how to use the graph. Edit away!

Randomize Custom Dialog

Finally, we cover the randomize custom dialog.

This lets you choose how your random gradients are generated.

The number of nodes corresponds to the number of distinct color bands there are in the gradient. The hue between controls what colors can show up between the nodes. The Saturation between is how dull or vibrant the colors get. The luminescence between controls how dull or bright the color will be. I don't know what hue blending does. Hitting the generate button will cause a random gradient to be generated.

So those are all the items on the process menu, and that's pretty much all you need to know to run Apomap. Now I shall hit upon some of the other options, in no particular order.

Window Options Menu

Take a look at the window options menu, shown below. This let you choose different ways of displaying all the gradients you have open. It is possible to have more than one gradient open at once. Go to file, new, and open a couple of new gradients. Then go back to the windows option and click on each of the menu items in turn to see how they arrange the gradients. They're really easy to figure out.

Now that you have multiple gradients open, it is possible to copy and paste between gradients. That's what these copy

and paste

buttons are for. These buttons will copy and paste the selected part of one gradient to the selected part of another gradient, as shown below.

These two buttons are not to be confused with the gradient copy

and gradient paste button.

Be careful, because it's easy to get them confused. The gradient copy and paste buttons copy and paste the entire gradient, and not just the selected part of it.

Next, there is a way to change the color of a single stripe at a time. Right-click on the gradient, as shown below. A set color option will show up. Select it.

The select color dialog will show up. Select a color, and the stripe will turn that color.

Finally, you can save your gradients by clicking the 'save' button,

or by selecting save under the file menu. Congrats! You now know how to use Apomap!
